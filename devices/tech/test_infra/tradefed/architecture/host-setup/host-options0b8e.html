Tradefed 主机选项

Tradefed (TF) 中的主机选项指的是在主机级别（一个 TF 实例）应用且影响自动化测试框架本身行为的选项。这些选项通常不会影响测试本身，而是支持自动化测试框架提供的操作。

主机选项定义了一系列选项，这些选项可用于自定义 TF 行为，如：

  • 可以同时刷写多少设备。这对资源有限的物理主机很有用。
  • 应将文件下载到的目录。如果任何原因导致默认临时文件夹不是正确的位置，可以对其进行更改。
