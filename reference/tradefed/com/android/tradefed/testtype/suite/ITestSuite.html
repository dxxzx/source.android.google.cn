ITestSuite

public abstract class ITestSuite
extends Object implements IRemoteTest, IDeviceTest, IMultiDeviceTest, IBuildReceiver, ISystemStatusCheckerReceiver, IShardableTest, ITestCollector, IInvocationContextReceiver, IRuntimeHintProvider, IMetricCollectorReceiver, IConfigurationReceiver, IReportNotExecuted, ITokenRequest

java.lang.Object
   ↳ com.android.tradefed.testtype.suite.ITestSuite
Known direct subclasses
BaseTestSuite, TfSuiteRunner
BaseTestSuite A Test for running Compatibility Test Suite with new suite system. 
TfSuiteRunner Implementation of ITestSuite which will load tests from TF jars res/config/suite/ folder. 
Known indirect subclasses
AtestRunner, TestMappingSuiteRunner
AtestRunner Implementation of ITestSuite  
TestMappingSuiteRunner Implementation of BaseTestSuite to run tests specified by option include-filter, or TEST_MAPPING files from build, as a suite. 

Abstract class used to run Test Suite. This class provide the base of how the Suite will be run. Each implementation can define the list of tests via the loadTests() method.
