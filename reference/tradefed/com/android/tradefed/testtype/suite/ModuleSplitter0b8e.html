ModuleSplitter

public class ModuleSplitter
extends Object

java.lang.Object
   ↳ com.android.tradefed.testtype.suite.ModuleSplitter

Helper to split a list of modules represented by IConfiguration into a list of execution units represented by ModuleDefinition.

Each configuration may generate 1 or more ModuleDefinition depending on its options and test types:

  • A non-shardable IConfiguration will generate a single ModuleDefinition.
  • A shardable IConfiguration will generate a number of ModuleDefinition linked to the IRemoteTest properties:
    • A non - IShardableTest will generate a single ModuleDefinition.
    • A IShardableTest generates one ModuleDefinition per tests returned by IShardableTest.split().
